# Qt GUI Framework

## Qt

- C++ 기반 GUI APplication 개발시 널리 쓰이는 프레임워크
- 다른 플랫폼으로 Porting이 자유로움
  - One framework
  - One codebase
  - Any platform
- Windows, Linux, MacOS, Android, VxWorks, Tizen 지원
- 네트워크, 그래픽, DB, OpenGL 등 쉬운 API 지원
