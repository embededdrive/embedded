# 네트워크 보안

- 가용성, 기밀성, 무결성에 대한 공격 및 장애로부터 모든 컴퓨터 리소스를 보호하는 것

## 위험

- 컴퓨터 바이러스
- 해킹
- 랜섬웨어
- 악성코드
- DDoS

## 해결책

- 개인 사용자
  - 암호 복잡도 강화
  - 보안 업데이트
  - 방화벽
  - 안티바이러스
  - VPN
  - 안전한 인터넷
  - 스팸 필터링
  - WiFi 보안
- 기업
  - 보안 정책 수립
  - 방화벽 구성
  - IDS / IPS 도입
  - VPN 구성
  - 암호화
  - 보안 솔루현 업체 위탁

## 제로 트러스트 모델

- 장점
  - 서브넷의 트래픽 감소로 네트워크 성능 향상
  - 네트워크 오류 해결 능력 향상
  - 세분화로 로깅 및 모니터링 프로세스 간소화
  - 침해 탐지 시간 단축
